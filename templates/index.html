Place Order REST API

Order Endpoints:

  • GET /orders - Returns the list of all orders.
  • GET /order/order_number - Returns a specific order.
  • POST /order/order_number - Add an order by the order number.
  • PUT /order/order_number - Add or Update an order by the order number.
  • DELETE /order/order_number - Delete an order by the order number.

Item Endpoints:

  • GET /items - Returns the list of all orders.
  • GET /order/product_code - Returns a specific order.
  • POST /order/product_code - Add an order by the product code.
  • PUT /order/product_code - Add or Update an order by the order number.
  • DELETE /order/product_code - Delete an order by the order number.
